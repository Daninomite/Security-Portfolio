\# TryHackMe Write-Up: Search Engine OSINT \& Security Tools

\*\*Date:\*\* March 27, 2026

\*\*Platform:\*\* TryHackMe

\*\*Topic:\*\* Open Source Intelligence (OSINT) \& Security Research Tools



\## Objective

Learn how to leverage search engines and specialized security 

tools for open source intelligence gathering and security research.



\## Key Concepts Covered



\### Advanced Search Engines

Most people use search engines at a surface level, but security 

analysts use advanced search techniques to find specific 

information quickly and efficiently. Learning to search 

effectively is an underrated but critical skill in cybersecurity.



\### Tools Introduced



| Tool | What It Is | Why It Matters |

|---|---|---|

| Shodan | A search engine for internet connected devices | Lets analysts find exposed devices, servers, and vulnerabilities |

| CVE Database | Common Vulnerabilities and Exposures list | The standard reference for known security vulnerabilities |

| Exploit Database | Archive of public exploits | Used to research known attack methods for specific vulnerabilities |

| VirusTotal | Scans files and URLs for malware | Used to quickly check if something is malicious |



\## What I Learned



\- Advanced search techniques are a seriously underrated skill 

&#x20; in cybersecurity — knowing how to find information quickly 

&#x20; is just as important as knowing technical concepts

\- Shodan exposes just how many devices are connected to the 

&#x20; internet with little to no security — eye opening!

\- CVEs are how the security community tracks and communicates 

&#x20; about known vulnerabilities — every security professional 

&#x20; needs to know how to read them

\- VirusTotal is a quick and free first step when investigating 

&#x20; a suspicious file or URL



\## Why This Matters in Cybersecurity

These tools are used daily by security analysts, penetration 

testers, and threat intelligence teams. OSINT (Open Source 

Intelligence) gathering is often the first step in both 

offensive security and defensive investigations. Knowing 

where to look for information is half the battle.



\## Reflection

It's surprising how much security relevant information is 

publicly available to anyone who knows where to look. These 

tools highlight why organizations need to actively monitor 

their own public facing exposure.
